What is KenOS?

KenOS is operating system that has it's own begining in the 2017, targeting x86 architectures by now. The idea is pretty straightforward - it should give Unix compatibility while maintaining own special features. It came in highschool, when I started to be interested in programming more and more. The whole OS is based on C/C++ with gcc/binutils crosscompiler toolchain, in addition docker and python is used for managing various stuff like automatic compiling with github actions, discord paravirtualizer (ssh-like way of communication with VM), et cetera.

Comes with x86 compatibility

Aims to be Unix-like

Targets to have own package-manager
